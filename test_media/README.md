# 测试素材（本地下载，不提交 Git）

本目录用于存放 **30s~1min** 的真实视频测试素材，覆盖质量模式自动化/人工测试所需场景。

## 重要说明

- **版权/许可**：只使用明确可用的公开许可素材（本目录会记录来源与许可）。
- **不提交 Git**：视频体积较大，默认被 `.gitignore` 忽略；只保留本 README。
- **生成方式**：推荐用 `automation/scripts/fetch_test_media.py` 一键下载+生成变体。

## 素材清单（建议最小集，对应文档 S1~S5）

运行脚本后会生成（文件名可能含时间戳，以下为默认名）：

- **S1-普通素材（无硬字幕）**：`test_media/S1_clean.mp4`
- **S2-有硬字幕素材（用于擦字幕/字幕位置盒）**：`test_media/S2_hardsub.mp4`
- **S3-长句/快语速素材（用于读速/断行压力）**：`test_media/S3_fast.mp4`（去静音+加速，制造更连续语流）
- **S4-多说话人/对白素材（用于上下文压力）**：`test_media/S4_dialog.mp4`（分段变调模拟“多人说话”）
- **S5-极端噪声素材（用于鲁棒性/门禁）**：`test_media/S5_noise.mp4`（明显加大噪声强度）

另外（可选）：

- **S0-静音素材（验证空字幕不崩）**：`test_media/S0_silent.mp4`

## 来源与许可（脚本默认）

默认下载源：

- **视频**：Blender Foundation 公布的开源短片/预告片（例如 Sintel），许可通常为 **CC-BY 3.0**（以源页面为准）。
- **中文音频**：LibriVox 的中文朗读（示例：朱自清《匆匆》），通过 Internet Archive 提供下载链接；LibriVox 录音通常声明为 **Public Domain（至少在美国）**。
- **正文段落**：脚本默认从中文音频的 **第 30 秒** 开始截取（可用 `--audio-start-s` 调整）。

- 你需要在对外分发测试素材时保留署名信息；仅内部测试通常问题不大，但也建议遵守署名要求。
- 脚本会在 `test_media/_sources.json` 中记录下载 URL 与生成参数，便于追溯。
