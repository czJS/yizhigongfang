# 本地视频中译英配音与字幕方案（当前实现）

## 目标
离线将中文视频转为英文配音 + 英文字幕（可选双语），使用 whisper.cpp + Marian 翻译 + Coqui TTS。

## 当前技术栈
- ASR：`whisper.cpp` + ggml 模型（现用 `ggml-medium-q5_0.bin`，低配可用 small）
- 翻译：`Helsinki-NLP/opus-mt-zh-en`（Marian，小模型，CPU 可用）
- TTS：Coqui TTS（模型 `tts_models/en/ljspeech/tacotron2-DDC` + vocoder `hifigan_v2`，CPU 运行）
- 媒体处理：`ffmpeg`（提取音频、复合音视频、封装字幕）

## 目录结构（建议）
```
project/
  assets/models/
    ggml-small.bin                # whisper.cpp 模型
    en_US-amy-low.onnx            # piper 备用模型（当前未用）
```

```bash
# Python 包（已使用）
```

## 模型准备
- ASR：`assets/models/ggml-small.bin`（已就绪）
- 翻译：首次运行自动缓存 `Helsinki-NLP/opus-mt-zh-en` 到 `~/.cache/huggingface/hub`
- TTS：首次运行自动缓存 Coqui 模型到 `~/Library/Application Support/tts/...`（已完成缓存）

## 运行示例（当前配置）
```bash
cd /Users/chengzheng/Desktop/译制工坊
export PATH="$HOME/Library/Python/3.9/bin:$PATH"
HF_ENDPOINT=https://hf-mirror.com \
python3 scripts/asr_translate_tts.py \
  --video 测试视频.mp4 \
  --output-dir outputs/test_run \
  --whispercpp-bin /usr/local/opt/whisper-cpp/bin/whisper-cli \
  --whispercpp-model assets/models/ggml-small.bin \
  --tts-backend coqui \
  --coqui-model tts_models/en/ljspeech/tacotron2-DDC \
  --coqui-device cpu \
  --bilingual-srt
```

输出位置（示例）
- `chs.srt` / `eng.srt` / `bilingual.srt`
- `tts_full.wav`
- `output_en.mp4`（英文配音）
- `output_en_sub.mp4`（英文配音 + 英文字幕）

## 流程要点
1) ffmpeg 提取音频（16k，单声道）
2) whisper.cpp ASR 输出分段文本与时间戳
3) Marian 翻译生成英文字幕
4) Coqui TTS 分段合成 → 时长对齐 → 拼接成英文音频
5) ffmpeg 复合视频与新音轨，并封装字幕

## 注意事项
- 首次运行会下载翻译/TTS 模型，请保持网络；之后可离线复用缓存。
- 若需更快/更轻，ASR 可换更小 ggml；TTS 可换其他 Coqui 英模。
- 已规避 piper 的 dylib 问题，当前默认使用 Coqui TTS。若切回 piper，需确保相关 dylib 放置正确并设置 `DYLD_LIBRARY_PATH`。
