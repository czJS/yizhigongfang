## 1. 软件概述

`秒译出海视频译制软件`是一款面向视频译制、字幕本地化和内容出海场景的桌面应用软件。软件以桌面端为操作入口，结合本地业务后端和云端授权服务，为用户提供视频素材导入、字幕识别、字幕翻译、英文配音生成、音视频合成、字幕擦除与重烧录、规则维护、任务管理和登录授权控制等能力。

软件主要面向需要将中文视频快速处理为英文字幕或英文配音成片的用户，适用于内容出海、短视频剪辑、课程本地化、宣传视频译制、批量视频加工等业务场景。软件整体强调“低门槛、可批量、可管理、可交付”的产品定位。

---

## 2. 开发目的

本软件的开发目的主要包括以下几个方面：

1. 提供一套可用于视频译制和内容出海的桌面化处理工具，降低用户在字幕提取、翻译、配音和成片输出过程中的人工操作成本。
2. 将原本需要分散完成的语音识别、字幕编辑、翻译、配音、视频合成等步骤整合到统一工作流中，提升处理效率。
3. 支持批量化视频处理，满足多个素材连续处理和统一交付的需求。
4. 在保证基础可用性的前提下，通过规则中心、质量检查、任务审核、历史记录和系统资源页等能力增强产品可控性和可维护性。
5. 通过云端登录与激活机制对商业化授权进行统一管理，使桌面产品具备可交付、可授权、可续期、可管理的商业闭环能力。

## 5. 软件总体架构

本软件采用“桌面前端 + 本地业务后端 + 云端授权服务”的分层结构。

### 5.1 桌面前端

桌面前端基于 Electron 和 React 实现，负责为用户提供统一的图形化操作界面。其主要职责包括：

- 接收用户上传或选择的视频素材
- 提供模式选择、任务创建、交付设置和任务查看界面
- 展示系统状态、历史任务、审核结果和规则管理界面
- 与本地后端接口进行通信
- 与云端授权服务交互，处理登录、激活、续期和设备限制提示

### 5.2 本地业务后端

本地业务后端基于 Python 和 Flask 实现，负责调度视频处理流程和管理任务运行。其主要职责包括：

- 提供统一的本地 API 接口
- 读取系统配置和硬件状态
- 启动任务、查询进度、获取日志和产物
- 管理术语表、规则中心和审核相关数据
- 调用语音识别、翻译、语音合成和音视频处理流程
- 对任务创建与恢复操作进行授权校验

### 5.3 云端授权服务

云端授权服务负责账号登录和授权状态控制，主要能力包括：

- 邮箱验证码登录
- 激活码兑换
- 授权状态查询
- 设备绑定与解绑
- 授权到期、冻结、续期控制
- 管理后台支持激活码生成和授权管理

## 6. 主要功能模块说明

### 6.1 登录与授权模块

软件提供统一的登录与激活入口。用户可在桌面端输入邮箱、验证码和激活码完成登录与授权。系统支持授权状态控制，包括未激活、正常可用、已到期、已冻结等状态，并在受限状态下提示用户输入新激活码恢复使用。

主要功能包括：

- 邮箱验证码登录
- 激活码输入与兑换
- 当前授权状态查询
- 到期后的应用内恢复
- 设备数量限制与设备管理
- 安装包版型与授权版型匹配校验

### 6.2 模式选择模块

软件提供模式选择页，用于向用户展示不同工作模式的定位与适用场景。当前产品重点承载轻量模式，并保留更高阶模式的展示入口与资源提示能力。

主要功能包括：

- 显示可用模式
- 显示不同模式的处理定位
- 根据资源条件控制模式可用性
- 保存默认模式偏好

### 6.3 素材导入与任务创建模块

软件提供向导式任务创建流程，支持用户批量导入视频文件并按顺序串行处理。用户可以在创建过程中设置任务名称、输出目录、审核开关以及字幕区域相关参数。

主要功能包括：

- 批量导入视频素材
- 任务列表排序与移除
- 设置输出目录
- 保存任务基础参数
- 启动批量处理流程

### 6.4 字幕区域设置模块

软件提供视频预览和字幕区域设置能力，支持用户对原片硬字幕的擦除区域和新字幕的放置区域进行可视化调节。该模块可帮助用户在含硬字幕素材中快速完成字幕覆盖和重烧录设置。

主要功能包括：

- 视频预览
- 字幕区域矩形定位
- 擦除方式选择
- 字幕字号调节
- 字幕显示区域参数保存

### 6.5 视频处理与任务调度模块

本模块负责驱动视频处理主链路。用户启动任务后，系统将按照既定顺序完成素材处理，并在任务中心持续显示处理状态、当前阶段、异常情况和交付目录。

主要功能包括：

- 任务启动与排队
- 运行状态轮询
- 日志读取
- 失败后重新开始
- 支持取消和恢复
- 交付目录打开与产物导出

### 6.6 审核与规则中心模块

软件提供规则中心和审核相关能力，以便用户对识别结果、固定用词和替换规则进行统一维护，并对任务处理中间结果进行进一步检查和修正。

主要功能包括：

- 全局识别纠错规则维护
- 英文替换规则维护
- 审核抽屉查看
- 中英字幕读写与差异查看
- 术语与规则文件上传

### 6.7 历史任务与交付模块

本模块负责展示历史任务、处理结果和交付目录信息，便于用户追踪任务执行过程和输出结果。

主要功能包括：

- 查看历史任务
- 查看单任务详情
- 打开交付目录
- 导出诊断包
- 清理中间产物

### 6.8 系统状态与资源模块

系统页用于展示本地后端状态、硬件能力、资源包状态和运行时信息，帮助用户了解当前安装环境是否满足软件使用要求。

主要功能包括：

- 查看本地后端是否可用
- 查看 CPU、内存、显卡和加速状态
- 查看模型资源是否就绪
- 查看本地服务状态
- 刷新运行状态和重启后端

## 7. 主要业务流程说明

### 7.1 登录与激活流程

1. 用户打开桌面软件。
2. 软件展示登录与激活一体页。
3. 用户输入邮箱地址并请求验证码。
4. 用户输入验证码和激活码。
5. 云端授权服务校验登录信息和激活码合法性。
6. 校验成功后返回用户信息、授权状态和设备信息。
7. 桌面端进入主工作台。

### 7.2 批量任务创建流程

1. 用户进入新建任务向导。
2. 用户选择一个或多个视频文件。
3. 用户设置任务名称、输出目录和审核开关。
4. 用户根据需要调整字幕擦除与放置区域。
5. 用户确认任务参数并启动处理。
6. 系统将任务加入队列并按顺序执行。

### 7.3 视频处理与交付流程

1. 本地后端接收任务启动请求。
2. 系统读取配置和硬件状态。
3. 系统执行视频识别、字幕处理、翻译、配音、合成等步骤。
4. 处理过程中持续更新任务状态、阶段和日志。
5. 处理完成后生成字幕、音频、视频等产物。
6. 用户在任务中心或历史页查看并打开交付目录。

### 7.4 授权受限与恢复流程

1. 当软件检测到授权到期或冻结时，前端进入只读限制状态。
2. 用户仍可查看系统页、历史页和账号相关信息。
3. 用户可在应用内输入适用于当前安装包的新激活码。
4. 云端服务重新校验授权状态。
5. 校验通过后恢复任务处理能力。

## 9. 软件的技术特点

本软件具有以下技术特点：

1. 采用 Electron 桌面端与 Python 后端协同架构，兼顾易用性和处理能力。
2. 通过本地 API 统一连接前端界面与任务处理能力，使功能模块之间具备清晰的分层结构。
3. 支持批量任务串行处理，便于长视频或多素材连续执行。
4. 提供字幕区域可视化设置能力，增强对带硬字幕视频的适配性。
5. 提供规则中心、术语维护、审核抽屉和质量检查等能力，提升内容输出的一致性和可交付性。
6. 通过本地后端与云端授权服务双重控制任务入口，提高授权控制的可靠性。
7. 支持系统状态检测和资源就绪检查，便于用户在不同环境中快速定位问题。
