# 打包文档入口

这份目录专门放“如何产出可交付安装包/压缩包”的说明。

## 支持的平台

- **Windows**：✅当前已完整支持（安装包 + models_pack + ollama_pack）
- **macOS**：⚠️预留文档入口（如要支持，需要补充对应脚本与签名/公证流程）

## 文档入口

- **Windows**
  - `docs/质量模式/交付与Windows打包.md`

- **macOS**

  - （可选）`docs/打包/macos/打包机准备清单.md`

- Windows：`packaging/windows/scripts/`
- macOS：`packaging/macos/scripts/`（预留）
