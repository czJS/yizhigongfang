# 本地中译英配音桌面端（GUI）需求说明（macOS / Windows）

## 1. 目标与范围
- 离线运行：导入视频 → 提取音频 → ASR（whisper.cpp）→ 翻译（Marian/M2M/NLLB）→ TTS（Coqui）→ 混流字幕/音频导出。
- 跨平台：macOS（Intel/Apple Silicon）与 Windows 10+；无管理员权限也可安装和运行。
- 首次可用：安装包中预置必需二进制与模型（ASR ggml、MT、Coqui TTS），无网络也可完成首个任务。
- 角色：产品可配置参数与模型；研发按此开发前后端、打包与分发。

## 2. 角色与用户故事
- 普通用户：选择视频，点击「开始」，查看进度，导出英文配音视频与字幕；可切换模型档位（基础/中端/高端）。
- 产品/运营：配置默认模型与参数、输出目录策略、日志与错误提示文案；管理预置模型包。
- 研发：实现前端 UI、后端服务、任务管线、模型加载、本地存储与打包。

## 3. 功能需求
3.1 任务流程  
- 导入视频（拖拽/选择），显示基本信息（时长、分辨率、音轨）。  
- 硬件检测：启动时检测 CPU 核数、内存、GPU（显存），将用户分档（普通/中端/高端），写入状态栏。  
- 参数面板：  
  - ASR：模型选择（small/medium ggml）、线程数、VAD 开关/阈值、去噪开关。  
  - 翻译：模型选择（Marian/M2M/NLLB）、设备（CPU/GPU 自动）、可选润色（开关）。  
  - TTS：后端选择（Coqui 默认）、模型选择、设备（CPU/GPU 自动）、分句/去重开关。  
  - 字幕：是否生成双语、输出格式（srt/vtt/mov_text）。  
  - 输出：输出目录、文件命名规则。  
- 模型推荐：根据硬件档位自动给出推荐组合（可一键应用，支持手动修改）。  
- 运行与进度：显示阶段进度（1/7…7/7）、耗时、剩余预估；可取消任务。  
- 中间产物编辑：支持查看/编辑中文+时间戳、英文+时间戳，修改后作为下一阶段输入（继续翻译/继续 TTS）。  
- 结果展示：播放预览（视频+字幕）、查看英文/双语字幕、打开输出目录。  
- 日志与诊断：显示/复制运行日志，提示常见错误（缺模型、路径错误、权限不足等）。  
- 模型管理：检测模型是否存在；若缺失可提示导入本地模型包；显示体积与更新时间；清理缓存。  
- 任务队列（可选 V1.1+）：支持多个任务排队执行。

3.2 配置（产品可控）  
- 默认模型与参数（按机器档位预设，见下），可一键应用推荐。  
- 输出目录与命名模板。  
- 是否允许切换高级模型（大体积）和 GPU 使用。  
- 错误提示与引导文案。  
- 是否启用日志上报/本地收集（默认仅本地）。

3.3 非功能性  
- 离线可用；无网络亦可完成一次完整任务。  
- 资源占用可控：基础档 CPU-only 可完成 1 分钟视频在 ~10–20 分钟内（视模型而定）。  
- 稳定性：任务取消不中断其他运行；异常时保留日志。  
- 安全：仅本地文件访问；模型和日志存储在用户目录。

## 4. 技术方案（概述）
- 前端：Electron + React，提供参数面板、进度、日志、结果预览。  
- 后端：Python + Flask，本地 HTTP API；进度/日志通过 WebSocket 推送（无则轮询）。  
- 任务执行：后端启动子进程运行 `asr_translate_tts.py`，按任务生成独立输出目录。  
- 预置与加载：安装包内附带 FFmpeg、whisper.cpp 可执行，以及预置模型目录；运行时通过参数指向本地路径。  
- 日志：后端标准输出写日志文件，前端实时显示；任务完成后提供打开日志按钮。

## 5. 平台与打包
### macOS
- 产物：.app + .dmg；内含 FFmpeg、whisper.cpp、预置模型目录（models/）。  
- 路径：  
  - 可执行：`app/Contents/Resources/bin/`  
  - 模型：`app/Contents/Resources/models/`（ASR ggml、MT、Coqui TTS 缓存）  
  - 缓存/输出：`~/Library/Application Support/<AppName>/`（用户可在设置中改输出目录）。  
- 权限/签名：建议代码签名和公证，减少 Gatekeeper 拦截；若包含动态库（如 piper）需处理隔离属性。  
- 无管理员安装：支持拖拽安装到 Applications；模型随包提供，无需额外下载。

### Windows
- 产物：NSIS 安装包；随包放置 FFmpeg、whisper.cpp、模型目录。  
- 路径：  
  - 可执行：`<install_dir>\\bin\\`  
  - 模型：`<install_dir>\\models\\`  
  - 缓存/输出：默认 `C:\\Users\\<user>\\AppData\\Local\\<AppName>\\`，可配置。  
- 签名：建议代码签名减低 SmartScreen 拦截。  
- 无管理员模式：选择用户目录安装；避免写系统目录。

## 6. 模型与资源预置
- ASR：whisper.cpp ggml 模型（small、medium-q5_0），放 `models/asr/`。  
- MT：Marian（Helsinki-NLP/opus-mt-zh-en）基础必备；可选 M2M100/NLLB 目录放 `models/mt/`。  
- TTS：Coqui TTS 模型及声码器预下载到 `models/tts/coqui/`，运行时设置 `TTS_HOME` 指向该目录。  
- FFmpeg、whisper.cpp 可执行放 `bin/`。  
- 体积策略：提供“完整版”（含模型）与“精简版”（不含模型，支持手动导入）。  
- 校验：随包提供校验和，启动时快速校验模型存在性与大小。

## 7. 配置与默认参数（建议）
- 硬件分档（自动检测 + 手动切换）：  
  - 普通：CPU 4–8 线程，内存 8–16GB，无独显。  
  - 中端：CPU 8–12 线程，内存 16–32GB，GPU 显存 4–8GB。  
  - 高端：CPU ≥12 线程，内存 ≥32GB，GPU 显存 ≥8–12GB。  
- 推荐模型组合（单一方案，可一键应用，允许微调）：  
  - 普通：ASR `ggml-small-q5_0`；MT `Marian`；TTS `Coqui tacotron2-DDC + hifigan_v2`（CPU）；分句上限 50 字，去重开启，VAD 关闭。  
  - 中端：ASR `ggml-medium-q5_0`；MT `Marian`（保持较佳体验）；TTS `Coqui tacotron2-DDC`（GPU 优先，GPU 不可用则 CPU）；分句上限 50 字，去重开启，VAD 开启。  
  - 高端：ASR `ggml-medium-q5_0`（如显存≥12GB 可换 `large-v3 q5`）；MT `NLLB-200-distilled-600M`（单一高端方案）；TTS `Coqui` GPU（预置高质量 VITS）；分句上限 40–50 字，去重开启，VAD 开启。  
- 输出采样率：16k；分句阈值默认 50（高端可 40–50）；重复去重开启；双语字幕可选。

## 8. 中间产物与可编辑性
- 中间文件：  
  - 中文+时间戳（ASR 结果，json/srt）  
  - 英文+时间戳（翻译结果，json/srt）  
- 编辑能力：在 UI 中可查看并逐条编辑文本与时间戳（或导出后再导入），编辑后的结果作为下一阶段输入：  
  - 仅字幕：直接导出/复合  
  - 继续 TTS：以修改后的英文+时间戳重新合成  
- 版本管理：编辑后生成新版文件（带版本号），保留原始文件。

## 9. 界面要点（产品/设计参考，细化）
- 左侧导航栏：  
  - 工厂：导入、进度、中间产物、结果。  
  - 配置：参数设置、模型管理、缓存/线程/并发、语言偏好、硬件检测结果与推荐档位一键应用。  
- 工厂（主流程区）：  
  - 导入：文件选择/拖拽、文件信息。  
  - 进度：阶段进度条、耗时/剩余、日志折叠面板。  
  - 中间产物：中文/英文字幕列表，可内联编辑文本/时间码，支持导入/导出 json/srt。  
  - 结果：播放器（视频+字幕）、按钮（打开目录/复制路径/查看日志/导出字幕）。  
- 配置区：模型管理（检测/导入/清理）、缓存大小、线程/并发、语言偏好、硬件检测结果与推荐档位一键应用。  
- 错误提示：缺模型、磁盘不足、无权限、显卡不可用、文件路径包含非 ASCII 时的提示。

## 10. 研发实现指引
- 后端接口（示例）：  
  - POST /tasks/start {video, params} → task_id  
  - GET /tasks/{id}/status → 阶段、进度、耗时  
  - POST /tasks/{id}/cancel  
  - GET /tasks/{id}/log → tail  
- 进程管理：子进程运行 Python 脚本，传入本地模型路径；捕获 stdout/stderr，写 run.log。  
- 路径参数：前端传入模型/输出路径；后端注入 TTS_HOME、HF_ENDPOINT（可选离线）。  
- 错误处理：子进程非零退出时，前端展示日志摘要与建议。  
- 打包：Electron-builder / Tauri bundler，拷贝 bin/ 与 models/ 目录到资源路径；安装后保持可写输出目录在用户空间。  
- 测试：  
  - 离线模式：断网验证可完整跑通。  
  - 双平台冒烟：导入/运行/导出、日志、取消、路径含空格/非 ASCII。  
  - 性能：1 分钟视频基础档耗时预估、内存占用监测。

## 11. 风险与对策
- 包体积大：区分完整版/精简版，提示手动导入模型；可选在线下载。  
- 动态库兼容（mac 尤其 piper/espeak）：优先 Coqui；如需 piper，随包带依赖并签名。  
- 资源占用高：提供档位预设与线程上限；默认选择中档配置。  
- 磁盘占用：模型管理与清理入口。  
- Gatekeeper/SmartScreen：签名、公证，避免未签名依赖。

## 12. 交付物
- 可执行安装包（mac .dmg/.app，Win NSIS/MSIX），含预置模型/二进制。  
- 配置样例：默认参数与档位预设文件。  
- 使用说明：离线运行指引、模型路径说明、常见错误与解决。
