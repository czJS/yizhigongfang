# 本地中译英配音可视化界面技术评估（macOS / Windows）

## 目标与范围
- 提供离线可用的桌面图形界面，一键完成：导入视频 → ASR → 翻译 → TTS → 复合字幕/音频 → 导出。
- 支持 macOS（Apple Silicon/Intel）与 Windows 10+；尽量减小安装包体积，兼容无管理员权限安装。
- 便于后续扩展：模型选择、参数调优、日志与错误提示、进度与耗时统计。

## 架构建议
- 前端 UI：跨平台框架（Electron 或 Tauri），展示进度、日志、参数面板、任务队列、结果预览。
- 后端服务：Python 进程承载现有流水线（`asr_translate_tts.py`），通过本地 IPC/HTTP 调用。
- 任务执行：后端启动子进程执行流水线，写入工作目录；前端轮询/订阅进度（管道日志或 WS）。
- 模型与二进制：按平台预置/首次下载缓存（whisper.cpp ggml、MT 模型、Coqui TTS），FFmpeg 随应用打包或引导安装。

## 框架对比
| 维度 | Electron | Tauri | PyQt/PySide |
| --- | --- | --- | --- |
| 体积 | 大（>80–100MB） | 小（~10–20MB） | 中等（~50–100MB，含 Qt 库） |
| 技术栈 | Web 前端+Node | Web 前端+Rust 壳 | 纯 Python/Qt |
| 与现有 Python 集成 | 通过本地 HTTP/IPC 最简单 | 同上；Rust bridge 可选 | 直接调用，但 UI 开发成本高 |
| 分发/签名 | 成熟 | 较新但成熟度提升 | 成熟 |
| 推荐 | 若团队前端资源充足 | 追求体积小可选 | 希望全 Python 可选 |

建议：优先 Electron（最快落地、生态成熟）；若强烈追求体积，评估 Tauri。

## 关键技术点
- IPC/调用方式：前端通过本地 HTTP (127.0.0.1) 调用后端，后端暴露启动/查询任务接口；日志流可用 WebSocket 或轮询。
- 任务管理：每个任务生成唯一工作目录，保存中间产物与最终输出；支持取消、重试。
- 平台二进制：
  - FFmpeg：随应用打包（mac .app 内、Win 可执行同目录），或首次启动下载。
  - whisper.cpp：按架构预置可执行文件与 ggml 模型（small/medium）；可在设置中切换。
  - Coqui TTS：首次运行自动下载模型到用户缓存；需打包 `pydub` 依赖。若改回 piper，则需随包放 espeak-ng 依赖（mac 需处理签名和隔离）。
- 模型管理：提供“检查/下载/更新”入口，展示大小与剩余磁盘；支持替换 MT 模型 (Marian/M2M/NLLB) 与 ASR 模型 (small/medium)。
- 进度与耗时：后端标准输出解析阶段标记（1/7…7/7），前端显示百分比与预计剩余时间。
- 日志与诊断：保存 run.log；前端可一键复制日志；常见错误给出指引（缺模型/缺 FFmpeg/显卡不可用）。
- 资源检测：启动时检测 CPU 核心数、可用内存、是否有 GPU，给出推荐模型配置。
- UI 功能点：文件拖拽导入、参数面板（模型、线程数、VAD/去噪、翻译模型、TTS 选择、分句/去重开关）、输出路径选择、任务列表、播放预览、字幕下载。
- 国际化：UI 文案中英双语，可后续扩展。

## 分发与签名
- macOS：产出 .app + dmg，设置沙盒外本地文件访问；建议代码签名与公证以降低 Gatekeeper 拦截。
- Windows：产出 NSIS/EXE 或 MSIX；如有条件建议代码签名。
- 无管理员模式：尽量使用用户目录安装与模型缓存，避免写入系统目录。

## 安全与离线
- 默认离线；若需下载模型可配置镜像/HF_ENDPOINT。
- 禁止外部网络调用（除模型下载）；前端仅连接本地后端。

## 风险与缓解
- 体积/依赖大：Electron 方案包体大；可通过 Tauri 减少体积。模型单独下载避免膨胀。
- 动态库问题（mac piper/espeak-ng）：优先使用 Coqui TTS（纯 Python）；若需 piper，需随包携带兼容的 libespeak-ng 并签名。
- 运行时超时/卡顿：提供任务取消与日志；进度细分；对长句分句和去重避免 TTS 过长。
- 磁盘占用：模型缓存可视化管理与清理。

## 推荐落地方案（首版）
- 前端：Electron + React/Vue，简洁参数面板 + 进度/日志视图。
- 后端：现有 Python 脚本包装为本地 HTTP 服务（Flask/FastAPI），提供启动/状态/取消接口。
- 打包：mac .app + dmg，Win NSIS；随包带 FFmpeg、whisper.cpp 可执行文件（small+medium ggml），MT/TTS 等模型可直接预置在 models/ 目录，首次运行无需再下载（也可提供精简包+手动导入模式）。
- 功能 MVP：导入、运行、查看进度、播放结果、导出字幕；设置页管理模型/缓存与线程数；错误提示。

## 后续可扩展
- 任务队列与批处理。
- 模型市场/配置预设（基础/中端/高端）。
- 自动检测网络/带宽选择镜像或断点续传。
- 可选 TTS 模型切换（VITS 系列、piper 稳定语速版）。
