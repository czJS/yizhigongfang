## Desktop UI Smoke Checklist（每一步拆分后回归）

> 目标：确保重构拆分“只搬家不改行为”。每完成一个小步（移动文件 / 抽组件 / 抽 hook）就跑一遍。

### 启动与初始化
- 打开桌面端 UI，等待初始化完成。
- 观察：不会持续弹出“初始化失败/后端启动中”的错误循环；`config/hardware/health` 能正常获取（UI 有正常可用的模式/预设）。

### 云端登录与授权
- 打开桌面端后，云端登录门禁能正常展示，不白屏、不闪退。
- 输入邮箱后点击“发送验证码”，能触发请求；真实发信或 `dev_code` 联调提示至少应有一种可用。
- 输入验证码 + 激活码后点击“登录并进入”，能正常进入主界面。
- 激活码错误时：展示错误提示，不应残留脏登录态。
- `expired` 账号：允许进入只读界面，但“新建任务 / 继续处理”应被拦截，并能看到“输入新激活码”入口。
- `frozen` 账号：允许查看账号信息，但不能新建或继续处理任务，且提示联系管理员。

### 新建批次向导（Wizard）
- 添加视频（至少 1 个），能出现在任务列表。
- **lite 模式**下：支持批量添加视频，最多 10 个；列表顺序可调整，执行时应按顺序串行处理。
- 不选择输出文件夹也能继续（会提示需要手动下载交付物）。
- 创建批次：能创建并跳转到任务中心/或继续新建（确认对话框行为正常）。

### 队列与任务调度（Workbench）
- 对一个批次点击“开始/加入队列”，能进入 running/queued 状态。
- 暂停队列：提示出现，且“当前任务继续跑、完成后不自动进入下一个”的语义不变。
- 继续队列：能恢复调度。
- 取消当前任务：能发送取消请求并给出提示（不要求后端一定成功，但 UI 不崩）。

### 任务抽屉（Task Drawer）
- 从任务列表打开抽屉，默认内容正常渲染。
- 切换 tab 不崩（质量/日志/审校等）。
- 日志：能加载或至少出现“加载中/失败提示”，UI 不应卡死。
- Artifacts：列表能显示；下载链接点击不应导致前端报错（是否能实际下载取决于后端）。

### 审校流程（Review）
- 审校 tab 能打开并渲染表格/编辑器。
- 点击“重新生成/继续/应用审校”类按钮：会触发 loading/提示；即便后端失败，UI 不崩溃且能提示错误。

### 规则中心（Rules Center）
- 打开规则中心页面。
- 模板列表能刷新/展示（请求失败时有提示且不崩）。
- 表格编辑：新增/删除/修改条目不崩（是否保存成功取决于后端）。

### 系统页（System）
- license 弹窗能打开、输入 CDKEY、点击验证能触发请求与提示（失败也不崩）。
- 模型导入弹窗：在模型未就绪时会出现；点击“选择模型包并导入”会触发流程（失败有提示）。

### 授权后台（Admin）
- 输入后台管理密钥后，用户列表 / 激活码列表 / 设备列表能正常加载。
- 生成激活码成功后，列表刷新正常。
- 冻结 / 恢复 / 延期 / 解绑操作：成功时有反馈，失败时有提示，页面不崩溃。
